My Work

Projects

Dog-Nasium

This project, built using React, depicts a fictional business known as Dog-Nasium.
The concept of Dog-Nasium is a giant indoor dog park in which customers pay for different levels of access. Dog-Nasium offers 3 package plans that extend from 1 month, 6 months, and up to a year.

Dog-Nasium

The package plans within the park site also offer a customer the ability to add discounted access to satellite businesses within the park complex, as well as the affordance to donate any dollar amount to a dog rescue. Built using REACT with Local JSON database.
